## 六、实验总结
- 通过本次实验，我掌握了 HTML 文档的基本结构和创建方法，熟悉了常用 HTML 标签（如标题、段落、链接、列表等）的使用方式及其属性的作用。理解了 HTML 注释的重要性，它能有效提高代码的可读性和可维护性。同时，了解了 DOCTYPE 声明的作用，认识到不同 HTML 版本的声明差异，以及 HTML5 在简化声明方面的改进。
- 实验过程中发现，浏览器对标准 HTML 的解析具有较好的一致性，但不同 DOCTYPE 声明可能导致浏览器使用不同的渲染模式，这提示在实际开发中应正确声明文档类型，以确保页面的一致性显示。
